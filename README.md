# 社团管理系统 java747

## 项目概述

社团管理系统是基于Spring Boot和Vue构建的前后端分离平台，支持超级管理员、普通管理员和普通用户三种角色。系统功能全面，能够满足各类社团的管理需求。

## 系统角色

### 超级管理员

- 首页概览
- 用户管理
- 社团管理
- 社团类型管理
- 社团成员管理
- 社团活动管理
- 通知信息管理
- 入团申请管理
- 费用管理

### 普通管理员

- 首页概览
- 社团信息管理
- 社团成员管理
- 入团申请管理
- 社团活动管理
- 通知信息管理
- 费用管理

### 普通用户

- 首页概览
- 社团信息浏览
- 社团活动查看
- 入团申请
- 费用查询

## 功能模块

### 额外功能

- 用户登录与注册
- 个人信息修改
- 密码修改

## 技术栈

- **后端**: Java, Spring Boot, MyBatis-Plus
- **前端**: Vue
- **数据库管理系统**: MySQL
- **开发工具**: IntelliJ IDEA, JDK 1.8, Maven, Node.js (版本14及以上)

## 运行环境

- 后端: Java 1.8+环境的兼容性
- 前端: 现代浏览器环境
- 数据库: MySQL

## 部署步骤

1. 确保环境已经安装Java JDK 1.8、MySQL和Node.js（版本14及以上）。
2. 使用IntelliJ IDEA或兼容的IDE导入后端项目，并配置Maven构建。
3. 执行数据库脚本，初始化数据库结构。
4. 在IDE中运行后端应用程序或使用构建工具打包并部署。
5. 在前端项目目录下执行`npm install`安装依赖，随后执行`npm run build`编译项目。
6. 将编译后的前端文件部署到Web服务器。
